No 1 About Project

A Unity 3d Mobile AR Android application developed using Vuforia AR technology where we use an Image target to display the hoop. The ball can be thrown by swiping on the screen and the score will be shown on the screen. It works on all Android mobile phones.

It has feedback animation and also sound effects. I did it as an experiment during my internship to check how Vuforia works with Image Marker and when you show your camera on an Image target the basketball hoop appears and you can swipe to throw the ball into the hoop.
The following image is the target image and you should point your camera on the image.

No 2 The Process

The game is an infinte basketball hoop shooting game. It is a casual game which gives you sound and visual feedback when you score a ball.

No 2.1Programming

  • Added Image target prefab from the Vuforia plugin and updated the image to the stone image. Made the objects that need to appear as the child of the Image target
  • Ball Controller
    All the functions from getting the input and applying force to the ball is done in this script.
    The score updation is also done in this script using the collider function. Visual Feedback is also done inside the collider function

No 2.2Game Design

  • Game Manager is present to check the exit function and Audio Manager is used to hold all the sound and thus plays the sound of ball hitting the board and ground, and also crowd cheer sound when the ball gets inside the hoop.
